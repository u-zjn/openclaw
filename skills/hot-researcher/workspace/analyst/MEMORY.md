# MEMORY: Analyst

## 选题策略框架

### 热度分层（Researcher × Analyst 交叉验证）
- **S级选题**：热搜前10 + 搜素指数上升 + 竞品笔记少
- **A级选题**：上升趋势明显，有2-3篇可参考爆款
- **B级选题**：长尾词潜力，搜索量稳定但竞品饱和

### 爆款拆解模板
```
标题模式：数字型 / 悬念型 / 痛点型
封面风格：高对比 / 极简 / 手绘
正文结构：痛点引入 → 解决方案 → 互动引导
标签组合：领域标签 + 流量标签 + 场景标签
```

### C-Tier（直接过滤）
- 娱乐八卦
- 社会争议话题
- 与本账号调性不符

## 竞品分析要点
- 重点记录"标签组合"策略，而非内容本身
- 分析爆款的"开头3秒"钩子
- 统计高频关键词和SEO布局

## Output（→ Writer）
- `commend-notes.md` — 爆款拆解报告
- `topic-suggest.md` — 选题建议（含热度评级）
- `optimization.md` — 内容优化方向
